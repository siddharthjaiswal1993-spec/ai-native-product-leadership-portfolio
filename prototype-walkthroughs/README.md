## How to Use These Walkthroughs

Each walkthrough includes:
- **Demo goal** — one sentence on what you're showing
- **Problem** — what the prototype demonstrates about product thinking
- **Primary persona** — who this product is for
- **Loom script** — scripted narration for a 3-minute demo recording
- **Click path** — numbered steps through the prototype
- **Key screens** — 3-5 screen descriptions with what to highlight
- **If asked "is this real?"** — honest, confident answer
- **Interview talking points** — what to say about this project in a PM interview
